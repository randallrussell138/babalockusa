Some of our Features

Unlock methods 4 WAY authentication

There are four ways to open the door: fingerprint, card key, password, and emergency key.

Remote control anytime, anywhere Smartphone App compatible

Real-time door lock status can be checked inside and outside the house, and remote control and monitoring are possible with a mobile phone.

For visitors One-time password

By setting a one-time password in advance, you can open the door with a one-time password when a friend or relative visits you. Once the door is opened, the one-time password number is deleted and must be set again.

Stop worrying about outside! Life waterproof function

It works normally even in an exposed environment. It can be used with confidence unless it is exposed to direct rain.
